# mapcat-mapview-init

### Build dependencies
* node (min _v6.9.5_)
* npm (min _4.3.0_)

### Install Node.js dependencies
`npm install`

### Build
Release: `npm run build-min`  
Debug: `npm run build-dev`

### Usage
`<script src="mapcatview-min.js"></script>`
